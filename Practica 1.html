  • ¿Cómo funciona una computadora?

La computadora es un dispositivo tecnologico, Existen computadoras de escritorio, portátiles, servidores, supercomputadoras, entre otros, cada uno con características y funcionalidades específicas.

Caracteristicas

  • Una computadora se caracteriza por ser un sistema lógico complejo, versátil y que combina componentes electrónicos (hardware) y lógicos (software).
  • El Hardware incluye componentes físicos como la tarjeta madre, la unidad central de procesamiento (CPU), la unidad de procesamiento gráfico (GPU), la memoria RAM, dispositivos de almacenamiento (HDD/SSD), teclado, ratón, monitor, etc.
  • El Software Engloba el sistema operativo, aplicaciones, programas y herramientas que permiten la ejecución de tareas específicas.
